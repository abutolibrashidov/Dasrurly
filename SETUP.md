# 🚀 Tez Boshlash (Quick Start)

## ⚡ 1 Minutada Ishga Tushirish

### Talab Qilinadigan Dasturlar
- Node.js 16+ ([Download](https://nodejs.org))
- Ikki terminal oynasi (Command Prompt yoki PowerShell)

### Qadamlar

#### Backend (Terminal 1)
```powershell
cd c:\Users\KSM\Desktop\OSHXONA\backend
npm install
npm start
```

#### Frontend (Terminal 2)
```powershell
cd c:\Users\KSM\Desktop\OSHXONA\frontend
npm install
npm run dev
```

### Qo'llash
1. Taraychi oching: `http://localhost:5173`
2. 2 ta tab/oyna oching
3. Birida "Ofitsiant Paneli" bosing
4. Ikkinchisida "Oshxona Paneli" bosing

### Sinov Buyurtmasi
- **Ofitsiant**: Stol 1, Plov + Manti tanlang → Yuborish
- **Oshxona**: "Tayyorlashni Boshlash" bosing → "Tayyor" bosing
- **Ofitsiant**: Bildirishnoma olasiz! ✅
