## 🎁 BONUSLAR

### Rang Kodlari
```
🟠 YANGI      = Orange    = #FF9800
🔵 TAYYOR...  = Ko'k      = #2196F3
🟢 TAYYOR     = Yashil    = #4CAF50
```

### Ovoz Signallari
```
📢 DING-DING      = Yangi buyurtma (Oshxonada)
🎵 DO-RE          = Buyurtma yuborildi (Ofitsiantda)
🔔 BEEP-BEEP      = Tayyor bildirishnomasi (Ofitsiantda)
```

### Sanator
```
[3 Yangi]  ← Yangi buyurtmalar soni
[1 Tayyor] ← Tayyorlanayotgan soni
```
