## 🚀 QUICK START

### Installation (First Time)
```
1. Install Node.js 16+ from nodejs.org
2. Extract/download OSHXONA folder
3. cd OSHXONA/backend && npm install
4. cd OSHXONA/frontend && npm install
```

### Running (Every Time)
```
Terminal 1:
  cd OSHXONA/backend
  npm start
  → Server on http://localhost:3000

Terminal 2:
  cd OSHXONA/frontend
  npm run dev
  → Frontend on http://localhost:5173
```

### Testing (First Order)
```
1. Open http://localhost:5173 in 2 tabs
2. Tab 1: Click "Ofitsiant Paneli"
3. Tab 2: Click "Oshxona Paneli"
4. Tab 1: Enter table 1, select 2 items, click "Yuborish"
5. Tab 2: See new order, click "Tayyorlashni Boshlash"
6. Tab 1: Status changes to "Tayyorlanmoqda"
7. Tab 2: Click "Tayyor"
8. Tab 1: See notification "Stol 1 tayyor!" ✅
```
